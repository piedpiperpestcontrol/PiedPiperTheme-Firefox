# Pied Piper Pest Control Theme - Firefox

A London pest control browser theme for Mozilla Firefox.  

---

## Install Instructions (Temporary)

1. Download the [Firefox theme folder](./) or [ZIP file](#) from this repository.
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
3. Click **Load Temporary Add-onâ€¦**.
4. Select the `manifest.json` file in the folder.
5. Your theme will be applied immediately.

> For permanent use, install via [Firefox Add-ons store](https://addons.mozilla.org/).
